### Title
Zero Settlement Fee Market Creation via Pre-Default `touchMarket` Snapshot - (File: src/Midnight.sol / src/periphery/MidnightBundles.sol)

### Summary
`touchMarket` is permissionless and snapshots `defaultSettlementFeeCbp[market.loanToken]` at creation time with no check that defaults are non-zero. Because the `Midnight` constructor only sets `roleSetter` and leaves `feeSetter = address(0)`, any caller can invoke `touchMarket` (directly or via `buyWithUnitsTargetAndWithdrawCollateral`) before the `feeSetter` has configured defaults, permanently locking all seven settlement-fee breakpoints at zero for that market. Every subsequent `take` in that market accrues zero to `claimableSettlementFee[loanToken]`.

### Finding Description

**Code path and root cause**

`Midnight` constructor: [1](#0-0) 

`feeSetter` is `address(0)` at deployment; `defaultSettlementFeeCbp[token]` is the zero-initialized mapping default for every token.

`touchMarket` (permissionless `public`): [2](#0-1) 

Lines 777–784 unconditionally copy `defaultSettlementFeeCbp[market.loanToken]` into the new `MarketState`. There is no `require` that any of those seven values is non-zero, and no check that `feeSetter != address(0)` or that defaults have been configured.

`buyWithUnitsTargetAndWithdrawCollateral` calls `touchMarket` on line 64: [3](#0-2) 

**Exploit flow**

1. Attacker observes `Midnight` deployment. `feeSetter` is `address(0)`; `defaultSettlementFeeCbp[loanToken]` is `[0,0,0,0,0,0,0]`.
2. Before the `roleSetter` calls `setFeeSetter` and before the `feeSetter` calls `setDefaultSettlementFee`, the attacker calls `touchMarket(market)` (or `buyWithUnitsTargetAndWithdrawCollateral` with a valid `takes[0].offer.market`).
3. `touchMarket` creates the market and writes `settlementFeeCbp0..6 = 0` into `marketState[id]`.
4. Every subsequent `take` in this market computes: [4](#0-3) 
   With all cbp values zero, `settlementFee(id, timeToMaturity)` returns `0` for any TTM (the interpolation formula `(0 * ... + 0 * ...) / (end - start) = 0`). [5](#0-4) 
5. Because `_settlementFee = 0`, `buyerAssets == sellerAssets`, and: [6](#0-5) 
   adds zero to `claimableSettlementFee[loanToken]` on every take.

**Why existing checks fail**

- `touchMarket` has no access control and no minimum-fee guard.
- `setMarketSettlementFee` (the post-creation fix) requires `msg.sender == feeSetter` and `marketState[id].tickSpacing > 0` — the `feeSetter` can correct the market after the fact, but all takes executed before that correction paid zero fees and those fees are unrecoverable.
- The Certora rule `newMarketSettlementFeesMatchDefault` only verifies that market fees equal defaults at creation; it does not assert defaults are non-zero. [7](#0-6) 

### Impact Explanation
Any market created before `feeSetter` configures defaults for its `loanToken` will have all seven settlement-fee breakpoints permanently set to zero. Every `take` in that market pays zero settlement fee: `claimableSettlementFee[loanToken]` never accumulates for those takes, and the protocol permanently loses all settlement-fee revenue for the lifetime of that market (until `feeSetter` manually overrides via `setMarketSettlementFee`, but past takes are unrecoverable).

### Likelihood Explanation
The precondition is a deployment-ordering race: `feeSetter` is `address(0)` immediately after construction, and `defaultSettlementFeeCbp` is all-zeros for every token until explicitly configured. Any attacker monitoring the mempool can front-run the `setFeeSetter` + `setDefaultSettlementFee` transactions. The attack is repeatable for every new `loanToken` whose defaults have not yet been set, and requires no special privilege — only a valid `Market` struct with an allowed LLTV and sorted collateral params.

### Recommendation
Add a guard in `touchMarket` that requires all seven default settlement-fee breakpoints to be non-zero (or at minimum that `feeSetter != address(0)`) before allowing market creation:

```solidity
uint16[7] memory _defaultSettlementFeeCbp = defaultSettlementFeeCbp[market.loanToken];
require(
    _defaultSettlementFeeCbp[0] > 0 || feeSetter != address(0),
    DefaultFeesNotSet()
);
```

A stricter alternative is to require each breakpoint to be non-zero, or to require `feeSetter != address(0)` as a proxy for "the protocol has been fully initialized." Additionally, consider setting `feeSetter` in the constructor or adding an initialization function that must be called before any market can be created.

import "forge-std/Test.sol";
import {Midnight} from "src/Midnight.sol";

contract ZeroFeeMarketTest is Test {
    Midnight midnight;
    address loanToken  = makeAddr("loanToken");
    address collToken  = makeAddr("collToken");
    address oracle     = makeAddr("oracle");
    address attacker   = makeAddr("attacker");

    function setUp() public {
        midnight = new Midnight();
        // feeSetter is address(0); defaultSettlementFeeCbp[loanToken] == [0,0,0,0,0,0,0]
    }

    function testZeroFeeMarketCreation() public {
        CollateralParams[] memory cp = new CollateralParams[](1);
        cp[0] = CollateralParams({
            token: collToken,
            lltv: 0.77e18,
            maxLif: midnight.maxLif(0.77e18, 0.25e18),
            oracle: oracle
        });
        Market memory market = Market({
            loanToken: loanToken,
            collateralParams: cp,
            maturity: block.timestamp + 30 days,
            rcfThreshold: 0,
            enterGate: address(0),
            liquidatorGate: address(0)
        });

        // Attacker creates market before feeSetter sets defaults
        vm.prank(attacker);
        bytes32 id = midnight.touchMarket(market);

        // Assert all settlement fee breakpoints are zero
        uint16[7] memory fees = midnight.settlementFeeCbps(id);
        for (uint256 i = 0; i < 7; i++) {
            assertEq(fees[i], 0, "fee breakpoint should be zero");
        }

        // Assert settlementFee returns 0 for any TTM
        assertEq(midnight.settlementFee(id, 1 days),  0, "1d fee == 0");
        assertEq(midnight.settlementFee(id, 30 days), 0, "30d fee == 0");
        assertEq(midnight.settlementFee(id, 360 days),0, "360d fee == 0");

        // After N takes, claimableSettlementFee[loanToken] remains 0
        // (take setup omitted for brevity; assert claimableSettlementFee == 0 after each take)
        assertEq(midnight.claimableSettlementFee(loanToken), 0, "no fees accrued");
    }
}

    function touchMarket(Market memory market) public returns (bytes32) {
        bytes32 id = toId(market);
        if (marketState[id].tickSpacing == 0) {
            require(market.maturity <= block.timestamp + 100 * 365 days, MaturityTooFar());
            require(market.collateralParams.length > 0, NoCollateralParams());
            require(market.collateralParams.length <= MAX_COLLATERALS, TooManyCollateralParams());
            address previousCollateralToken;
            for (uint256 i = 0; i < market.collateralParams.length; i++) {
                address collateralToken = market.collateralParams[i].token;
                require(collateralToken > previousCollateralToken, CollateralParamsNotSorted());
                uint256 lltv = market.collateralParams[i].lltv;
                require(isLltvAllowed(lltv), LltvNotAllowed());
                require(
                    market.collateralParams[i].maxLif == maxLif(lltv, LIQUIDATION_CURSOR_LOW)
                        || market.collateralParams[i].maxLif == maxLif(lltv, LIQUIDATION_CURSOR_HIGH),
                    InvalidMaxLif()
                );
                previousCollateralToken = collateralToken;
            }

            MarketState storage _marketState = marketState[id];
            _marketState.tickSpacing = DEFAULT_TICK_SPACING;
            uint16[7] memory _defaultSettlementFeeCbp = defaultSettlementFeeCbp[market.loanToken];
            _marketState.settlementFeeCbp0 = _defaultSettlementFeeCbp[0];
            _marketState.settlementFeeCbp1 = _defaultSettlementFeeCbp[1];
            _marketState.settlementFeeCbp2 = _defaultSettlementFeeCbp[2];
            _marketState.settlementFeeCbp3 = _defaultSettlementFeeCbp[3];
            _marketState.settlementFeeCbp4 = _defaultSettlementFeeCbp[4];
            _marketState.settlementFeeCbp5 = _defaultSettlementFeeCbp[5];
            _marketState.settlementFeeCbp6 = _defaultSettlementFeeCbp[6];
            _marketState.continuousFee = defaultContinuousFee[market.loanToken];
            IdLib.storeInCode(market, INITIAL_CHAIN_ID);

            emit EventsLib.MarketCreated(market, id);
        }
        return id;
    }

    function settlementFee(bytes32 id, uint256 timeToMaturity) public view returns (uint256) {
        MarketState storage _marketState = marketState[id];
        require(_marketState.tickSpacing > 0, MarketNotCreated());

        if (timeToMaturity >= 360 days) return _marketState.settlementFeeCbp6 * CBP;

        // forgefmt: disable-start
        (uint256 start, uint256 end, uint256 feeLower, uint256 feeUpper) =
            timeToMaturity < 1 days   ? (  0 days,   1 days, _marketState.settlementFeeCbp0 * CBP, _marketState.settlementFeeCbp1 * CBP) :
            timeToMaturity < 7 days   ? (  1 days,   7 days, _marketState.settlementFeeCbp1 * CBP, _marketState.settlementFeeCbp2 * CBP) :
            timeToMaturity < 30 days  ? (  7 days,  30 days, _marketState.settlementFeeCbp2 * CBP, _marketState.settlementFeeCbp3 * CBP) :
            timeToMaturity < 90 days  ? ( 30 days,  90 days, _marketState.settlementFeeCbp3 * CBP, _marketState.settlementFeeCbp4 * CBP) :
            timeToMaturity < 180 days ? ( 90 days, 180 days, _marketState.settlementFeeCbp4 * CBP, _marketState.settlementFeeCbp5 * CBP) :
                                        (180 days, 360 days, _marketState.settlementFeeCbp5 * CBP, _marketState.settlementFeeCbp6 * CBP);
        // forgefmt: disable-end

        return (feeLower * (end - timeToMaturity) + feeUpper * (timeToMaturity - start)) / (end - start);
    }

rule newMarketSettlementFeesMatchDefault(env e, Midnight.Market market, uint256 index) {
