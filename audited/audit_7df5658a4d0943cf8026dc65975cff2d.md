### Title
Permissionless market creation with no-code `loanToken` causes permanent DoS on all loan-token-dependent operations - (File: src/libraries/SafeTransferLib.sol)

### Summary
`touchMarket()` validates collateral parameters, maturity, LLTV, and maxLif but performs no check that `market.loanToken` has deployed bytecode. `SafeTransferLib.safeTransferFrom` unconditionally enforces `token.code.length > 0` and reverts with `NoCode()`. Any market created with a no-code loan token address is permanently inoperable: every `take()`, `repay()`, and `withdraw()` call will revert, and the market state can never be unwound through normal protocol flows.

### Finding Description
**Root cause — missing code-existence check in `touchMarket()`:**

`touchMarket()` at `src/Midnight.sol:755-791` validates maturity, collateral count, sort order, LLTV tier, and maxLif, but never checks `market.loanToken.code.length > 0`. [1](#0-0) 

`SafeTransferLib.safeTransferFrom` at `src/libraries/SafeTransferLib.sol:24-25` unconditionally enforces the code-existence invariant before any call: [2](#0-1) 

**Exploit path:**

1. Attacker calls `touchMarket(market)` with `market.loanToken = address(1)` (or any address with no deployed code). All existing checks pass; the market is created and `marketState[id].tickSpacing` is set to `DEFAULT_TICK_SPACING`.
2. Any subsequent `take()` call reaches `src/Midnight.sol:455-456` and calls `SafeTransferLib.safeTransferFrom(offer.market.loanToken, ...)`. The `require(token.code.length > 0, NoCode())` guard fires unconditionally, reverting the entire transaction. [3](#0-2) 
3. The same `NoCode()` revert fires in `repay()` at line 520 and `withdraw()` at line 499, making every loan-token-dependent operation permanently impossible. [4](#0-3) [5](#0-4) 

**Why existing checks are insufficient:**

`touchMarket()` validates structural market parameters but has no guard on the loan token address itself. The collateral token addresses are validated only for sort order and non-zero value (enforced by `collateralToken > previousCollateralToken` where `previousCollateralToken` starts at `address(0)`), but `loanToken` has zero validation beyond being stored. [6](#0-5) 

**Clarification on state commitment:** The EVM atomically reverts all state mutations (positions, `totalUnits`, `claimableSettlementFee`) when `safeTransferFrom` reverts at line 455. There is no partial state commit. The DoS is not from corrupted state but from the market being structurally inoperable from creation.

### Impact Explanation
Any market created with a no-code `loanToken` is permanently unusable. `take()`, `repay()`, and `withdraw()` all revert with `NoCode()` on every call. If users supply collateral to such a market (which succeeds, since `supplyCollateral` uses the collateral token), they cannot borrow against it and the market cannot be used for its intended purpose. The market ID is permanently occupied in `marketState` with no recovery path. This is a concrete, irreversible DoS on the affected market.

### Likelihood Explanation
The precondition is trivially satisfiable: `touchMarket()` is permissionless, requires no stake or privilege, and the only constraint on `loanToken` is that it is stored. An attacker can call `touchMarket` with `loanToken = address(1)` in a single transaction at zero cost beyond gas. The condition is repeatable for any market parameter combination. Any user who then interacts with the market (e.g., supplies collateral expecting to borrow) is permanently locked out of loan-token operations.

### Recommendation
Add a `require(market.loanToken.code.length > 0, ...)` check inside the market-creation branch of `touchMarket()`, mirroring the same guard already present in `SafeTransferLib`. Optionally, apply the same check to each `collateralParams[i].token` for consistency.

```solidity
// Inside the `if (marketState[id].tickSpacing == 0)` block in touchMarket():
require(market.loanToken.code.length > 0, LoanTokenHasNoCode());
```

### Proof of Concept
```solidity
function testNoCodeLoanTokenDoS() public {
    // Build a structurally valid market except loanToken has no code.
    Market memory m;
    m.loanToken = address(1); // no deployed bytecode
    m.maturity = block.timestamp + 100;
    CollateralParams[] memory cp = new CollateralParams[](1);
    cp[0] = CollateralParams({
        token: address(collateralToken1),
        lltv: 0.77e18,
        maxLif: maxLif(0.77e18, 0.25e18),
        oracle: address(oracle1)
    });
    m.collateralParams = cp;

    // Step 1: market creation succeeds — no loanToken code check.
    bytes32 id = midnight.touchMarket(m);
    assertGt(midnight.tickSpacing(id), 0, "market created");

    // Step 2: take() always reverts with NoCode().
    Offer memory offer; /* ... populate valid offer fields ... */
    offer.market = m;
    vm.expectRevert(SafeTransferLib.NoCode.selector);
    midnight.take(offer, hex"", 1, address(this), address(this), address(0), hex"");

    // Step 3: repay() always reverts with NoCode().
    vm.expectRevert(SafeTransferLib.NoCode.selector);
    midnight.repay(m, 1, address(this), address(0), hex"");

    // Step 4: withdraw() always reverts with NoCode().
    vm.expectRevert(SafeTransferLib.NoCode.selector);
    midnight.withdraw(m, 1, address(this), address(this));
}
```

Expected assertions: all three `expectRevert(SafeTransferLib.NoCode.selector)` pass; the market is confirmed created but permanently inoperable.

    function touchMarket(Market memory market) public returns (bytes32) {
        bytes32 id = toId(market);
        if (marketState[id].tickSpacing == 0) {
            require(market.maturity <= block.timestamp + 100 * 365 days, MaturityTooFar());
            require(market.collateralParams.length > 0, NoCollateralParams());
            require(market.collateralParams.length <= MAX_COLLATERALS, TooManyCollateralParams());
            address previousCollateralToken;
            for (uint256 i = 0; i < market.collateralParams.length; i++) {
                address collateralToken = market.collateralParams[i].token;
                require(collateralToken > previousCollateralToken, CollateralParamsNotSorted());
                uint256 lltv = market.collateralParams[i].lltv;
                require(isLltvAllowed(lltv), LltvNotAllowed());
                require(
                    market.collateralParams[i].maxLif == maxLif(lltv, LIQUIDATION_CURSOR_LOW)
                        || market.collateralParams[i].maxLif == maxLif(lltv, LIQUIDATION_CURSOR_HIGH),
                    InvalidMaxLif()
                );
                previousCollateralToken = collateralToken;
            }

            MarketState storage _marketState = marketState[id];
            _marketState.tickSpacing = DEFAULT_TICK_SPACING;
            uint16[7] memory _defaultSettlementFeeCbp = defaultSettlementFeeCbp[market.loanToken];
            _marketState.settlementFeeCbp0 = _defaultSettlementFeeCbp[0];
            _marketState.settlementFeeCbp1 = _defaultSettlementFeeCbp[1];
            _marketState.settlementFeeCbp2 = _defaultSettlementFeeCbp[2];
            _marketState.settlementFeeCbp3 = _defaultSettlementFeeCbp[3];
            _marketState.settlementFeeCbp4 = _defaultSettlementFeeCbp[4];
            _marketState.settlementFeeCbp5 = _defaultSettlementFeeCbp[5];
            _marketState.settlementFeeCbp6 = _defaultSettlementFeeCbp[6];
            _marketState.continuousFee = defaultContinuousFee[market.loanToken];
            IdLib.storeInCode(market, INITIAL_CHAIN_ID);

            emit EventsLib.MarketCreated(market, id);
        }
        return id;
    }
